# JSON Frontend

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS

---

## Prerequisites

- Node.js 18+
- npm / yarn

---

## Running Locally
### 1. Clone Repository
git clone https://github.com/your-username/json-frontend.git
cd json-frontend

### 2. Install Depedencies
npm install

### 3. Copy local environment
cp .env.local.example .env.local

### Run Development Server
npm run dev
